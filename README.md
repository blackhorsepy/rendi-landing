# Estructura del proyecto

Este sitio quedó organizado para que sea más fácil ubicar cada archivo sin cambiar el funcionamiento principal.

## Carpetas

- `assets/images/`: imágenes usadas por la landing.
- `assets/icons/`: iconos y favicons del proyecto.
- `index.html`: página principal.
- `privacy.html`: página de política de privacidad.
- `terms.html`: página de términos de uso.

## Nota

`index.html` se mantiene en la raíz para no afectar despliegues estáticos simples ni enlaces existentes.
